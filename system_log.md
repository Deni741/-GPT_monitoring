## ✅ Checkpoint — Партнерська модель GPT_monitoring (липень 2025)

### 🔹 Архітектура:
- Основний Telegram-бот з GPT-ядром.
- 4 окремі трейдингові агенти:
  1. Swing Trader (теханаліз, RRR ≥ 1:3)
  2. StatArb (статистичний арбітраж)
  3. Sentiment Scanner (пампи, X/Twitter)
  4. Macro Tracker (фази ринку, капітал)

- Кожен агент може бути привʼязаний до окремого субакаунта на біржі (Bybit).
- Telegram-бот виступає як центральна платформа управління.

---

### 🔹 Рівні доступу для партнерів:
- `Whitelist`: доступ лише для перевірених ID або токенів.
- Ролі: перегляд, аналіз, копіювання, управління.
- Опціонально: Web dashboard (партнерський інтерфейс).

---

### 🔹 3 моделі взаємодії:
1. **📊 GPT-аналітика** — надсилання сигналів і звітів.
2. **🤖 Copy-Trading** — партнери повторюють угоди основного акаунта.
3. **💰 Pooled-Trading** — загальний акаунт, ділиться прибуток.
